# CSS Grid
This is CSS grid-based gallery.

## Motivation
The purpose of building this grid was to use the CSS grids functionality to create a photo gallery.

## Tech/framework used
<b>Built with</b>
- [HTML]
- [CSS]

## Future changes/added functions
1./ Each grid section will give a user the option to add a file path of an image they would like to use in the grid so that they can personalise the photo-gallery.

## Installation
To install this project simply click the green 'Code' button and download the ZIP file. Save the file to your desired directory and edit the contents in your favourite text editor. In order to edit the .scss files you will need to use a compiler to compile the main.scss to the main.css folder.
